# DatumDock 品牌资产

当前品牌主稿采用由项目名称直接构成的字标：左侧为浅橙色与浅蓝色交叠的 `DD` 单字母标记，右侧为深炭灰 `DatumDock` 名称。它比抽象图形更容易建立开源项目的识别度，也适合 GitHub、应用关于页和文档页的横向展示。

| `datumdock-wordmark-v2.png` | 1823 × 863、透明背景的上一版绿色主字标，保留以便回顾与比较。 |
| `datumdock-logo.png` | 1254 × 1254、透明背景的首版抽象草稿；暂不再作为主品牌引用，保留以便回顾与比较。 |

## 使用规则

- 在宽度充足的页面优先使用完整 `DD + DatumDock` 字标。
- 后续制作 Windows 应用图标、托盘图标和小尺寸 favicon 时，应单独从 `DD` 标记导出简化版本，而不缩小完整字标。
- 当前配色使用浅橙色、浅蓝色与炭灰，保持明亮、克制的莫兰迪视觉系统；深色主题和单色版本应作为独立资产发布。
- 每次调整都以新文件名保存，避免覆盖已使用的发布版本。

## English Summary

The current DatumDock brand uses a name-driven wordmark: a soft light-orange and light-blue interlocking `DD` monogram followed by the dark-charcoal `DatumDock` name. `datumdock-wordmark-v3.png` is the transparent primary asset for GitHub, documentation, and the application About page. Earlier wordmarks and the first abstract logo are retained as historical drafts. Derive a simplified icon from the `DD` mark for small Windows and tray contexts, and version future brand iterations instead of overwriting existing assets.
