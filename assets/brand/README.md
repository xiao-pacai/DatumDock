# DatumDock 品牌资产

当前品牌主稿采用由项目名称直接构成的字标：左侧为浅橙色与浅蓝色交叠的 `DD` 单字母标记，右侧为深炭灰 `DatumDock` 名称。它比抽象图形更容易建立开源项目的识别度，也适合 GitHub、应用关于页和文档页的横向展示。

| 文件 | 用途 |

| `datumdock-wordmark-v2.png` | 1823 × 863、透明背景的上一版绿色主字标，保留以便回顾与比较。 |
| `datumdock-logo.png` | 1254 × 1254、透明背景的首版抽象草稿；暂不再作为主品牌引用，保留以便回顾与比较。 |

## 使用规则

- 在宽度充足的页面优先使用完整 `DD + DatumDock` 字标。
- 后续制作 Windows 应用图标、托盘图标和小尺寸 favicon 时，应单独从 `DD` 标记导出简化版本，而不缩小完整字标。
- 当前字标保留浅橙色、浅蓝色与炭灰；应用现代视觉 v2 使用更清晰的品牌蓝、冷白/浅蓝灰表面和深色画布与其搭配。深色主题和单色 Logo 应作为独立资产发布。
- 每次调整都以新文件名保存，避免覆盖已使用的发布版本。

## English Summary

The current DatumDock brand uses a light-orange and light-blue interlocking `DD` monogram followed by the dark-charcoal name. Visual design v2 pairs it with a clearer brand blue, cool light surfaces, and a dark annotation canvas rather than the former Morandi-led application palette. `datumdock-wordmark-v3.png` remains the primary transparent asset; derive small icons from the DD mark and version future iterations instead of overwriting existing files.
